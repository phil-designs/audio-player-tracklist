# Audio Player with Tracklist

Tags: jQuery, javascript, php
Requires at least: 3.6.0
Tested up to: 6.9.1
License: GPL2

## Description

An audio player with tracklist functionality, built via a custom post type and inserted via shortcode. Requires Advanced Custom Fields (ACF).

## Tested on 
* Firefox 
* Safari
* Chrome
* Opera
* MS Edge

## Website 
http://www.phildesigns.com/

## Installation 
1. Upload ‘audio-player-tracklist’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Audio Players" custom post type and create timelines there
4. Use the shortcode to embed the player on your posts and pages, choose between mini and full layouts
